Aprendendo JSF pela apostila da K19

Dicas:

- Para alterar o encode padrão do jboss, adicione a linha abaixo no standalone.xml.
```
	<servlet-container name="default"  default-encoding="UTF-8">
```

- Para ver os facets que estão setados no projeto, abra:
```
	path_raiz_do_projeto/.settings/org.eclipse.wst.common.project.facet.core.xml
```

- Para ver o nome do projeto depois do deploy e outras propriedades, abra:
```
	path_raiz_do_projeto/.settings/org.eclipse.wst.common.component
```
